# Challenge 1-6

## Yêu cầu

- Node.js >= v20.18.3
- Yarn
- Metamask

## Chạy Local

\`\`\`bash

### Terminal 1: Local blockchain

yarn chain

### Terminal 2: Deploy contracts

yarn deploy

### Terminal 3: Start frontend

yarn start
\`\`\`

## Deploy lên Sepolia

\`\`\`bash

### Generate deployer address

yarn generate

### Check balance

yarn account

### Deploy

yarn deploy

## Verify contract

yarn verify --network sepolia
\`\`\`

## Deploy Frontend

\`\`\`bash
yarn vercel
\`\`\`

## Tests

\`\`\`bash
yarn test
\`\`\`
